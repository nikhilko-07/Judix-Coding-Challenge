Judix Coding Challenge
Project Overview

This project is a full-stack web application developed as part of the Judix Coding Challenge.
It demonstrates practical experience in frontend–backend integration, API handling, and clean project structuring using modern JavaScript frameworks.

The application follows a clear client–server architecture, ensuring scalability and maintainability.

Tech Stack

Frontend: Next.js, React, JavaScript

Backend: Node.js, Express

State Management: Redux Toolkit

API Communication: Axios

Styling: Tailwind CSS

Development Tools: Nodemon, ESLint

Project Structure
Judix-Coding-Challenge/
│
├── backend/   → Node.js backend server
└── client/    → Next.js frontend application

Setup & Run Instructions
Backend
cd backend
npm install
nodemon server.js


Backend runs on http://localhost:9000

Frontend
cd client
npm install
npm run dev


Update backend URL in:

client/src/config/index.js


Paste:

http://localhost:9000

Key Highlights

Clear separation of frontend and backend

REST API integration between client and server

Scalable project structure following industry practices

Clean, readable, and maintainable codebase

Purpose

This repository is submitted as part of a technical assessment to demonstrate full-stack development skills, real-world project setup, and professional coding standards.
